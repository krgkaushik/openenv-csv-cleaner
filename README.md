# Openenv Csv Cleaner
A project by Kaushik Gunjkar for the Meta/Scaler OpenEnv Hackathon.

## Description
This project implements a Reinforcement Learning (RL) environment using **FastAPI** and **Pydantic** to clean messy CSV data automatically. It identifies null values, formats dates, and fixes typos using LLM-based agents.

Check out the configuration reference at https://huggingface.co/docs/hub/spaces-config-reference
